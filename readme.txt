В этом учебном проекте доступ к проекту джанго реализован через переменные, с помощью библиотеки python-decouple и файл .env
для запуска вам понадобится создать файл .env, и в него скопировать:
SECRET_KEY=#здесь написать любой пароль (без пробела после равно и без кавычек)

войти в админку можно создав нового суперпользователя:
        python manage.py createsuperuser
        или с помощью логина и пароля admin: admin
